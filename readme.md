# MagicStick

MagicStick is a project to create a free flywheel pendulum on a stick that can stand alone as its own system, including batteries and all necessary electronics.

## Project Goals

- Build a self-contained flywheel pendulum system
- Integrate all power and control electronics onboard
- Enable wireless communication and sensing

## Planned Hardware

- **Motors:** 3x 4015 or 5010 brushless motors
- **Motor Controllers:** 3x Simple FOC controllers
- **Power:** 6x 18650 lithium-ion cells
- **Microcontroller:** 1x BLE Sense

## Status

Currently in the planning and prototyping phase.
